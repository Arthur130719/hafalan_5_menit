# Aplikasi Hafalan Cepat dalam 5 Menit

Aplikasi web sederhana untuk membantu menghafal kalimat dalam waktu 5 menit dengan teknik tampil-sembunyi berulang.

## Fitur
- Input custom kalimat yang ingin dihafal
- Siklus tampilkan kalimat selama 5 detik, lalu sembunyikan selama 5 detik
- Timer 5 menit dengan 15 siklus
- Antarmuka sederhana dan mudah digunakan

## Cara Menggunakan
1. Buka `index.html` di browser web
2. Masukkan kalimat yang ingin Anda hafal di textarea
3. Klik tombol "Mulai Hafalan"
4. Ikuti instruksi: lihat kalimat, lalu klik "Lanjut" saat tersembunyi
5. Ulangi sampai waktu habis

## Teknologi
- HTML
- CSS
- JavaScript

## Jalankan
Buka file `index.html` di browser web apa saja.
